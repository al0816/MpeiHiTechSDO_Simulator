Проект на Unity 3D, представляющий из себя авиационный симулятор
# Преимущества данной игры:
- Тащит на AMD
- Не тащит на Intel
- На моей пекарне овер9000 фпс

Игра имеет три киберспортивных режима, а именно
- CS mode
- Dota 2 mode
- Minecraft mode
